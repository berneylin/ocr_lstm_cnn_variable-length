## 依赖环境

Python 3.6.6

pip3 install freetype-py captcha pillow numpy

tensorflow v1.11.0

## 使用方法

编辑configs.py文件，更改相关属性。

运行gen_data_set.py，等待Training/Validation数据集生成完毕。

运行tf_model.py，等待模型训练完毕。
